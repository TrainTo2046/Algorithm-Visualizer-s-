Kruskal Algorithm pseudocode

T: O(ElogE)
S: O(N)

function union(x, y)
function find(x)
function maskeset(x):
  parent(x) = x
  rank(x) = 0

input: undirected grpah G = (V, E) with weight wE
output: MST defined by edges x

for u in V: makeset(u)
X = {}
sort edges E by weight
edge = {u, v} in E
for edge in increasing order of weight:
  if find(u) != find(v):
    X.add(edge)
    union(u, v)
