Union Find pseudocode

T: O(logN) -> height of tree
S: O(N)

Initialize with Parent[i] = i
Initialize with rank[i] = 0

function find(x):
  if Parent[x] != x:
    return find(Parent[x])
  else:
    return x

functin union(x, y):
  rx = find(x)
  ry = find(y)
  if rx == ry: return
  if rank(rx) > rank(ry):
    Parent[ry] = rx
  else:
    Parent[rx] = ry
    if rank(rx) == rank(ry): 
      rank(ry) = rank(ry) + 1
