Bellman-Ford pseudocode

T: O(V * E)

input: Direct graph G with edge length

dist = {}
n = # of nodes

for node in G:
  dist[node] = inf

start = s
dist[start] = 0

repeat (n - 1) times:
  for node in G:
    Edges = G[node]
    for edge in Edges:
      update(node, edge)



function update(node, edge):
  # curr minimum distance
  curr = dist[edge]
  # total distance from start to edge
  new = G[node][edge] + G[node]
  dist[edge] = min(curr, new)                                                                                                                             ))
