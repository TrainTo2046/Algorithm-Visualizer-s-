Algorithm Visualizer program

WELCOME!!!
This program was designed to help
you visualize various Algorithm to
teach you how it works and help
you learn new Algorithm.

This algorithms listed in this
program are the main ones that are
used in LC problems.

You can select an Algorithm,
Read about what they do
Read the code to the Algorithm
See how they work by clicking the
Play button

You can click on the Algo Visualizer
button to reset everything.
